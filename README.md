## Задание 2
Требуется реализовать кастомный аллокатор на языке С (не С++!).
Необходимо реализовать функции malloc и free.
Запрашивается выделение памяти только двух размеров — 15 байт и 180 байт.
Обратить внимание на быстродействие и эффективность работы с памятью.
Код должен работать на 8/16/32 битной архитектуре.
